# MasterPFE

## Présentation

Ce projet est un projet de fin d'études qui vise à développer une application moderne et professionnelle.

## Fonctionnalités

- Interface conviviale
- Performance optimisée
- Conformité avec les normes modernes

## Installation

1. Clonez le dépôt : `git clone https://github.com/YassinoMed/MasterPFE.git`
2. Installez les dépendances : `npm install`
3. Lancez l'application : `npm start`

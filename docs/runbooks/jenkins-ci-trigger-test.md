# Jenkins CI Trigger Test

Ce fichier sert de changement inoffensif pour valider le declenchement automatique du job `securerag-hub-ci` apres un `git push`.

## Usage

1. Modifier une ligne de ce fichier, par exemple la date du test.
2. Creer un commit.
3. Pousser vers `main`.
4. Verifier que Jenkins consomme bien le commit attendu.

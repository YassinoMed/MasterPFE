# Jenkins CD Trigger Test

Ce fichier sert de support documentaire pour les essais du job `securerag-hub-cd`.

Il permet de garder une trace simple des validations manuelles du pipeline CD sans toucher aux manifests Kubernetes ni aux services applicatifs.

## Usage

1. Lancer `securerag-hub-ci` et verifier la preuve de push.
2. Declencher `securerag-hub-cd`.
3. Mettre a jour ce fichier avec la date et le resultat observe.

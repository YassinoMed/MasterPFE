\paragraph{Lecture honnete.}
\begin{itemize}
  \item \textbf{TERMINE cote depot} : la chaine est concue, automatisee, versionnee et documentee.
  \item \textbf{DEPENDANT\_DE\_L'ENVIRONNEMENT} : la preuve runtime finale, la release finale, Kyverno live, la base externe et Jenkins live doivent encore etre executes ou rejoues sur la cible.
  \item \textbf{PRET\_NON\_EXECUTE} : la modernisation des secrets par SOPS, External Secrets Operator ou Vault reste preparee mais non operee.
  \item \textbf{PARTIEL} : la demonstration live reste incomplete tant que le cluster cible n'est pas rejoue proprement de bout en bout.
\end{itemize}
